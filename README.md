=====================

pip install PyPDF2

Usage: 
 slideshare.py -u <url> -f <output pdf  name>
	
    optional arguments:
    -h, --help  show this help message and exit
    -u ,--url   Type the URL of slideshare report
    -f ,--flile output report name

=======
